# Sobre el proyecto:

Sistema que tiene por objetivo poner en practica el uso de node, express y deployar aplicación en Railway.

```
Pasos para ejecutar el proyecto:
      1. git clone [url_repositorio_proyecto]
      2. npm install
      3. npm start

```

```
Tecnología usada:
      NodeJS
      Express
```

```
Dependencias:
      hbs
      dotenv
```
